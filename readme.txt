Este el el readme del proyecto

aqui vas a encontrar la informacion del proyecto 

Que es un proyecto sencillo pero un poco mas avanzado que el tipico proyecto de todo list o similares es un un sistema de punto de venta como los que se usan en tiendas de abarrotes aqui estoy practicando javascript para iniciar mi camino de desarollador web por ahora solo son archivos html y javascript vainilla nada destacable pero un gran avance para mi poco a poco mientras lo siga haciendo ire agregando mas cosas por ahora seria todo 


Funcionamiento:

Lo unico que llevo hecho es una tabla que se genera dinamicamente con algunos datos que estan en arreglos en el codigo (si te preguntas por que lee las cosas a destacar del proyecto) un inventario que despliega los productos asociados al provedor seleccionado y una "recepcion de provedores" que en base a una lista de los provedores cuando tu seleccionas un provedor te despliega los productos asociados a ese provedor 

Cosas a destacar:

no tiene base de datos ni local storage esto por que estoy practicando lo basico eventualmente a futuro si agregare base de datos o algo asi por ahora los datos estan en arreglos del codigo no es lo mejor pero es en lo que me ha funcionado para practicar 

El por que de este proyecto es que trabajo en una tienda de auto servicio y se me ocurrio hacer algo similar a lo que usamos en la tienda ire agregando las ideas que se me ocurran o vea en mi trabajo
